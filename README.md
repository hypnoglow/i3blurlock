# i3blurlock

Simple blur lock screen for i3. Works on top of i3lock.

## Install

1. Install dependencies:
```
    yaourt -S xkb-switch-git --noconfirm
```

2. Clone this repo

3. Build and install:
```
    makepkg -sfi
```
Or just Build:
```
    makepkg -sf
```
And install later using pacman:
```
    sudo pacman -U i3blurlock-*.tar.xz
```
